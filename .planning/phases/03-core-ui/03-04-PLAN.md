1. Create `frontend/src/components/dashboard/FullscreenToggle.tsx`:
   - 'use client' component
   - Props: `containerRef: React.RefObject<HTMLDivElement | null>`
   - State: `isFullscreen` (boolean), synced with `fullscreenchange` event on document
   - useEffect: add/remove `fullscreenchange` event listener to detect external fullscreen exits (e.g., Escape key)
   - Only render button if `document.fullscreenEnabled` is true (check in useEffect + state, since SSR has no document)
   - Use `Maximize` and `Minimize` icons from lucide-react
   - Button: shadcn Button variant="ghost" size="icon", positioned absolute top-2 right-2
   - onClick: if fullscreen, `document.exitFullscreen()`; else `containerRef.current?.requestFullscreen()`

2. Create `frontend/src/components/dashboard/DashboardErrorBoundary.tsx`:
   - 'use client' component
   - This is a FUNCTIONAL error state display, not a React class ErrorBoundary (since iframe errors don't trigger React error boundaries)
   - Props: `onRetry: () => void`
   - Renders: centered content with AlertTriangle icon from lucide-react, "Erro ao carregar dashboard" heading, "O dashboard nao pode ser carregado. Verifique sua conexao e tente novamente." description, and a "Tentar novamente" button (shadcn Button) that calls onRetry
   - Style: full container height, centered, muted background

3. Create `frontend/src/components/dashboard/DashboardEmbed.tsx`:
   - 'use client' component
   - Props: `url: string, title: string`
   - State: `loading` (true initially), `error` (false initially), `iframeKey` (number, for forcing reload)
   - Ref: `containerRef` for fullscreen target, `timeoutRef` for error timeout
   - On mount (useEffect):
     - Start a 30-second timeout. If iframe hasn't fired onLoad by then, set error=true, loading=false
     - Cleanup: clear timeout on unmount
   - iframe onLoad handler: clear timeout, set loading=false, error=false
   - Retry handler: increment iframeKey (forces iframe remount), set loading=true, error=false, restart timeout
   - Render structure:
     ```
     <div ref={containerRef} className="relative" style={{ height: 'calc(100vh - 8rem)' }}>
       {loading && <LoadingOverlay />}  // Centered spinner with "Carregando dashboard..." text
       {error && <DashboardErrorBoundary onRetry={handleRetry} />}
       {!error && (
         <iframe
           key={iframeKey}
           src={url}
           title={title}
           className="w-full h-full border-none"
           allowFullScreen
           onLoad={handleLoad}
         />
       )}
       <FullscreenToggle containerRef={containerRef} />
     </div>
     ```
   - The loading overlay should be absolutely positioned over the iframe (not replacing it) so the iframe can load in the background
   - Use z-index to layer: iframe z-0, loading overlay z-10, fullscreen toggle z-20
  </action>

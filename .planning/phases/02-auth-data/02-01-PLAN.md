    6. Create api/internal/service/auth.go:
       - Struct AuthService with cfg *config.Config field
       - Constructor: func NewAuthService(cfg *config.Config) *AuthService
       - Method: func (s *AuthService) VerifyAndLogin(ctx context.Context, googleToken string) (*model.AuthUser, string, error)
         - Calls idtoken.Validate(ctx, googleToken, s.cfg.GoogleClientID)
         - Extracts email, name, picture, sub from payload.Claims
         - Calls s.isAllowed(email) — returns fmt.Errorf("Email nao autorizado") if false
         - Creates JWT with claims: email, name, picture, sub, exp (24h), iat
         - Signs with HS256 using s.cfg.JWTSecret
         - Returns AuthUser + signed token string
       - Method: func (s *AuthService) isAllowed(email string) bool
         - Check exact match in AllowedEmails (case-insensitive)
         - Check domain match in AllowedDomains (case-insensitive)
         - Return true if either matches
       - Method: func (s *AuthService) ValidateSessionToken(tokenStr string) (jwt.MapClaims, error)
         - Parse JWT with HMAC verification using JWTSecret
         - Return claims if valid, error if not

    7. Create api/internal/handler/auth.go:
       - Struct AuthHandler with service *service.AuthService
       - Constructor: func NewAuthHandler(svc *service.AuthService) *AuthHandler
       - Method: func (h *AuthHandler) Login(w http.ResponseWriter, r *http.Request)
         - Decode LoginRequest from body
         - Call h.service.VerifyAndLogin(r.Context(), req.GoogleToken)
         - On error: writeError(w, 403, err.Error()) — errors are already in Portuguese
         - On success: writeJSON(w, 200, LoginResponse{Token: token, User: *user})

    8. Create api/internal/middleware/auth.go:
       - func Auth(jwtSecret string) func(http.Handler) http.Handler
         - Extract "Bearer " token from Authorization header
         - If missing: return 401 with {"error":"Nao autorizado"}
         - Parse JWT with HMAC verification
         - If invalid/expired: return 401 with {"error":"Token invalido ou expirado"}
         - Store claims in context: context.WithValue for "user_email" key
         - Call next.ServeHTTP

       Interface AuthContextType: {
         user: AuthUser | null,

         isLoading: boolean,
         token: string | null,
         login: (credentialResponse: { credential?: string }) => Promise<void>,
         logout: () => Promise<void>,
       }

       Helper function parseJwt(token: string): AuthUser | null
         - Decode base64url payload (same pattern as reference project)
         - Extract email, name, picture, sub
         - Return null on any error

       AuthProviderInner component:
         - State: user (AuthUser | null), token (string | null), isLoading (boolean, default true)
         - Computed: isAuthenticated = user !== null

         - useEffect on mount (session restore):
           a. Read 'gis_auth_token' from localStorage
           b. If exists, parseJwt to get user display info
           c. If parsed successfully, set user + token state
           d. Also call POST /api/session to refresh the httpOnly cookie (ensures cookie exists)
           e. Set isLoading = false (in finally block)

         - login callback (async):
           a. Extract credential from credentialResponse. If missing, return.
           b. Call apiFetch<{ token: string; user: AuthUser }>('/auth/login', { method: 'POST', body: JSON.stringify({ google_token: credential }) })
           c. If result.ok is false, throw new Error(result.error.message ?? 'Erro ao autenticar')
           d. Store result.data.token in localStorage as 'gis_auth_token'
           e. Call POST /api/session with { token: result.data.token } to set httpOnly cookie
           f. setUser(result.data.user), setToken(result.data.token)

         - logout callback (async):
           a. localStorage.removeItem('gis_auth_token')
           b. await fetch('/api/session', { method: 'DELETE' })
           c. setUser(null), setToken(null)

       AuthProvider component (outer wrapper):
         - Reads NEXT_PUBLIC_GOOGLE_CLIENT_ID from process.env
         - Wraps children in GoogleOAuthProvider with clientId
         - Inside GoogleOAuthProvider, renders AuthProviderInner

       Export useAuth hook:
         - Gets context, throws Error if undefined (same as reference)

                  setLoading(false)
